FirebaseWebAuthn

repository • demo

A Firebase Extension for authentication with WebAuthn passkeys, a new web standard for passwordless authentication. Allows websites built on Google’s app development and cloud platform to use biometric authentication in place of passwords while changing only one line of app code.

ShortcutsAPI

repository

An API and library of iOS shortcuts used to create highly detailed home automations based on the user's Focus mode, location, and time of day. This project addresses limitations in Apple’s HomeKit system which make it impossible to disable motion-activated lights when the user’s iPhone is in Sleep Focus, for example.
